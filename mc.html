Minecraft

Want to play on a Minecraft server where everyone treats everyone with respect, where everyone helps each other and run a simple survival server without too many mods? Look no further! Come join us! The server is ran on Java, but we have a plugin that lets Bedrock players join too!

(you need to wait for an admin or the owner to join in order to grant you permissions to build and similar)

To join on Java, use this: Address: mc.aridan.net

To join on Bedrock, use this: Address: mc.aridan.net Port: 25609
